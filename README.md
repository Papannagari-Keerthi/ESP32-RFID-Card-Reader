# ESP32 RFID Card Reader

## Overview

This project demonstrates RFID card detection using the ESP32 microcontroller and MFRC522 RFID module through SPI communication.

## Components Used

* ESP32
* MFRC522 RFID Module
* RFID Card / Tag
* Jumper Wires

## Features

* RFID card detection
* UID reading
* SPI communication
* Real-time serial monitoring
* Embedded systems implementation

## Working Principle

The ESP32 communicates with the MFRC522 RFID module using the SPI protocol. When an RFID card or tag is placed near the reader, the module reads the card UID and displays it through the serial monitor.

## Technologies Used

* ESP32
* Arduino Programming
* SPI Communication
* MFRC522 Library
* Embedded Systems
